# 🏦 ATM Management System

A beginner-friendly ATM Management System built using Python. This is a console-based application that simulates basic banking operations using dictionaries, loops, and conditional statements.

## 📌 Features

- ✅ Create a new account
- 💰 Deposit money
- 💸 Withdraw money
- 🔄 Transfer money between accounts
- 💳 Check account balance
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3
- Dictionaries
- Loops (`while`)
- Conditional Statements (`if-elif-else`)
- User Input

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/ATM-Management-System.git
   ```

2. Open the project folder.

3. Run the Python file:

   ```bash
   python atm_management_system.py
   ```
